# Comprehensive Content Creation for Sanity CMS

## Mission
Create production ready 3000+ word content for ALL blog posts, comparison posts, listicle posts, templates, and glossary terms in Sanity CMS. Content must be SEO optimized, human written quality, and ready for immediate publication.

## Critical Requirements

### Formatting Rules (STRICT)
- ❌ **NO hyphens** (use "and" instead of dash)
- ❌ **NO en-dashes** (—)
- ❌ **NO em-dashes** (–)
- ✅ Use **bullets** for lists
- ✅ Use **numbered lists** for steps/sequences
- ✅ Include **tables** for comparisons
- ✅ Use **short sentences** (max 20 words)
- ✅ Add **table of contents** for long articles
- ✅ Include **quotes** from experts
- ✅ Add **comparison sections**
- ✅ Include **review/rating elements** where applicable

### Content Quality Standards
- **Length**: Minimum 3000 words per piece
- **Style**: Human written, conversational, authoritative
- **SEO**: Optimized for Google and AI/LLM understanding
- **Structure**: Clear headings (H2, H3), logical flow
- **Value**: Actionable insights, practical examples
- **Engagement**: Questions, statistics, real world scenarios

### Step 3: Content Creation Code Pattern

```javascript
// Example portable text structure with ALL required elements

function createComprehensiveContent(type, slug) {
  return [
    // Table of Contents
    block('Table of Contents', 'h2'),
    ...numberList([
      'Introduction',
      'Main Section 1',
      'Main Section 2',
      // ... etc
    ]),

    // Introduction
    block('Introduction', 'h2'),
    block('First paragraph with context...'),
    block('Second paragraph building on first...'),
    block('Third paragraph with transition...'),

    // Main sections with tables
    block('Comparison Analysis', 'h2'),
    block('Below is a comprehensive comparison table:'),
    // Table structure
    {
      _type: 'table',
      _key: randomKey(),
      rows: [
        {
          _key: randomKey(),
          cells: ['Feature', 'Option A', 'Option B', 'Winner']
        },
        {
          _key: randomKey(),
          cells: ['Price', '$99/mo', '$129/mo', 'Option A']
        },
        // ... more rows
      ]
    },

    // Bullet lists
    block('Key Features', 'h3'),
    ...bulletList([
      'Feature one with detailed explanation',
      'Feature two with practical example',
      'Feature three with measurable benefit',
      // ... etc
    ]),

    // Numbered steps
    block('Implementation Guide', 'h3'),
    ...numberList([
      'Step one with clear instruction',
      'Step two building on previous step',
      'Step three with expected outcome',
      // ... etc
    ]),

    // Quotes
    block('According to industry expert John Smith: "Quote content here that adds credibility and authority to the article."', 'blockquote'),

    // Conclusion
    block('Conclusion', 'h2'),
    block('Summary paragraph wrapping up key points...'),
  ];
}
```
